Issue

        Gradle sync failed Connection timed out connect. If you are behind an HTTP proxy.....




Solution

        1. 设置Samsung Maven 库为代理例外  Apply；

            File  Settings...  Apearance & Behavior  System Settings  HTTP Proxy  勾选 Manual proxy configuration  No proxy for 10.252.




        2. 重启 Android Studio,  项目 gradle.properties, 如下：

            systemProp.http.proxyHost=109.130.240.146

            systemProp.http.nonProxyHosts=10.252.

            systemProp.http.proxyPort=8888




            再次 sync  successed!.




        3. 如果依然超时，确认是否已经申请：http10.252.250.538081lmcontentgroupspublic 的防火墙，

            如果没有，申请防火墙后重试。
